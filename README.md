

## Description

Church school Restful API 

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```

## Built With

* [Node.js](http://www.dropwizard.io/1.0.2/docs/) - The web framework used
* [Nest](https://github.com/nestjs/nest) framework TypeScript starter repository
* [MySQL](https://rometools.github.io/rome/) - Used to generate RSS Feeds
